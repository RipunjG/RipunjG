## About Me

I'm a developer and engineer passionate about bridging the gap between **software**, **data science**, and **computational engineering**. I enjoy building real-world applications — from full-stack collaboration platforms and ML pipelines to CFD simulations and voice-controlled systems.

- Exploring the intersection of **AI, fluid dynamics, and software engineering**
- Building tools that make data accessible without needing to write code
- Currently working on a **Voice-Controlled Music Recommendation System**
- I have a GitHub Pages portfolio at [RipunjG.github.io](https://RipunjG.github.io)
